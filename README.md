# 🚀 CAREER Guide – AI-Powered Career Roadmap Generator

## 📌 Overview

**CAREER Guide** is a smart web application that helps users design a personalized career roadmap based on their background, skills, and goals.

Instead of confusion and random advice, users get a **structured, actionable path** tailored specifically to them.

---

## 🎯 Problem It Solves

Many students and professionals:

* Don’t know what skills to learn next
* Feel lost about career direction
* Follow generic advice that doesn’t fit them

👉 This project solves that by generating **custom career guidance instantly**.

---

## 💡 Key Features

* 🧠 **AI-Based Roadmap Generation**
* 🎓 Input-based personalization (education, skills, goals)
* 📈 Career progression suggestions
* 🏢 Industry-specific guidance
* ⚡ Clean and modern UI

## 📋 How It Works

1. User fills out the form:

   * Education level
   * Current year
   * Career goal
   * Skills
   * Target industries

2. Clicks **“Generate My Roadmap”**

3. System:

   * Processes the input
   * Generates a structured roadmap

4. User is redirected to:
   👉  Dynamic page on which the whole roadmap according to your need

## ⚙️ Installation & Setup

### 1️⃣ Clone the repository

```bash
git clone https://github.com/your-username/career-guide.git
cd career-guide
```

### 2️⃣ Run the project

#### 👉 Option A: Without Backend

Just open:

```bash
index.html
```

#### 👉 Option B: With Flask Backend

```bash
pip install flask
python app.py
```

Then open:

```
https://careerguide-2y8r.onrender.com
```
